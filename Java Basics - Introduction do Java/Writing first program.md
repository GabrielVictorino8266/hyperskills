# Writing first program
Vamos aprender sobre um templete básico de todo programa Java a ser construído.


```java
public Class Main{
    public static void main(String[] args){
        System.out.println("Hello, World");
    }
}
```

## Terminologia básica
- Programa: sequência de instruções.
- Statement: ação única, um comando. (finalizado por ;)
- Block: grupo de statement, normalmente entre {}
- Method: sequência de statements
- Syntax: regras para escrever o código
- Keyword: uma palavra reservada já para a linguagem
- Identifier or name:
- Comment: comentário para explicação
- Whitespace: caracteres não visível
