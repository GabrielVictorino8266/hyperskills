# Basic literals: numbers, strings and characters
Entendemos por Java Literal como qualquer valor atribuído diretamente para uma variável. Assim, entramos nos tipos de dados que o Java permite. Os 4 principais são listados aqui:

## Inteiros
Os números inteiros como conhecemos, são basicamente, números considerando valores negativos, zero e números positivos.
```java
int numApples = 100;
```

Ainda podemos melhorar a legibilidade, caso necessário, fazendo da seguinte forma:

```java
int numPackedApples = 1_000_000;
```

## Caracteres
Quando falamos de caracateres, estamos falando de um símbolo, uma letra. Exemplo, uma letra, um dígito, um espaço em branco.

```java
char charOne = '1';
int numOne = 1;
```
**OBS: com caracteres, ainda podemos fazer uma operação matemática com eles.**

## String (Conjunto de Caracateres)
Quando falamos de uma sequência, como uma palavra, começamos a ver a String. Exemplo:

```java
String firstString = "Hyperskill é uma plataforma fantástica.";
// Observe abaixo a diferença
String stringA = "A";
char charA = 'A';
```
